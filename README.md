## Research Projects

### Artificial Intelligence Research Project

This research paper explores the rapid growth of Artificial Intelligence (AI), its applications across multiple industries, ethical concerns, advantages, limitations, and future impact on society. The project demonstrates my research skills, analytical thinking, and ability to communicate technical concepts using credible academic sources and APA formatting.

### Data Mining Assignment

This assignment examines the principles of data mining, including techniques used to discover patterns within large datasets. The project discusses practical applications of data mining in business, healthcare, and technology while demonstrating research and analytical writing skills.

### Decision Tree Machine Learning Analysis Using Scikit-Learn

This project analyzes a Decision Tree classification model developed using Python and the Scikit-Learn library. The paper explains the libraries used, the Iris dataset, how the model works, the generated visualizations, and why the algorithm represents supervised machine learning.
